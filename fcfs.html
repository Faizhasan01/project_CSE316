FCFS

It is the simplest of all the Disk Scheduling Algorithms...

Advantages

  1. No indefinite delay.
  2. No starvation.

Disadvantages

  1. Not the best service.
  2. Disk time is not optimized.

Algorithm

  1. Request array contains indexes...
  2. Tracks are addressed one by one...
  3. Total seek count is incremented.
  4. Repeat until all tracks are addressed.
